# Introdução
O script Julia-Packages.jl instala de forma automatica uma série de pacotes no
Julia necessários para a utilização do Lamind.

# Lista de pacotes
  - Atom
  - Cbc
  - Clp
  - CUTEst 
  - GR
  - IJulia
  - ImageMagick
  - Images
  - Ipopt
  - JuMP 
  - Plots
  - PyPlot
  - Qwt
  - Rdatasets
